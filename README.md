# 🪨📜✂️ Rock Paper Scissors

A simple SwiftUI game that challenges your brain to **win or lose** at Rock, Paper, Scissors.

## 🧠 Game Concept

Each round:

- The app shows its move: Rock, Paper, or Scissors.
- You're told to either **win** or **lose** against it.
- Pick the correct move based on the instruction.

You'll earn points for correct answers and play through 8 rounds!

## ✨ Features

- Clean and colorful UI using SwiftUI and gradients.
- Dynamic game logic with random instructions.
- Score tracking and final results alert.
- Responsive layout and large emoji buttons for easy play.

## 🛠 Tech Stack

- Swift
- SwiftUI
